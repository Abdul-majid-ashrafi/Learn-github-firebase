Firebase Hosting

  1. Document of firebase hosting
  2. run on cmd npm install -g firebase-tools
  3. run on cmd firebase login
  4. create a project on Firebase
  5. go to out of root directory
  6. open cmd
  7. run firebase init
  8. run firebase deploy
